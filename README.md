# RAG_CHATBOT
RAG chatbot built with LangChain to query "Finance Bill 25/26" PDF using OpenAI's gpt-3.5-turbo. Processes PDF with PyPDFLoader, splits into token-based chunks, and stores embeddings in ChromaDB with all-MiniLM-L6-v2. Optimized for free-tier quota with minimal tokens and robust error handling.
